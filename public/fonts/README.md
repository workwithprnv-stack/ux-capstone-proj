# CRONOS Custom Fonts

To complete the installation of **Acme Gothic** and **Avenir Next**, please place your licensed webfont files (`.woff2`) in this directory.

### Expected Files:
- `acme-gothic-bold.woff2` (Mapped to Acme Gothic Bold)
- `avenir-next-regular.woff2` (Mapped to Avenir Next Regular)
- `avenir-next-medium.woff2` (Mapped to Avenir Next Medium)
- `avenir-next-bold.woff2` (Mapped to Avenir Next Bold)

*Until these files are added, the application will use Montserrat and Inter as high-fidelity geometric fallbacks.*
